Physical Design
The process of transforming a logical netlist into a physical layout that can be manufactured, including placement and routing.
Floor Plan
Creating the physical layout blueprint of the chip, defining block placement and I/O locations.
Power Plan
Designing the power distribution network that delivers stable voltage to all parts of the chip.
Placement
Optimally positioning standard cells on the chip to minimize wire length and improve performance.
CTS
Clock Tree Synthesis for balanced clock distribution across all sequential elements in the design.
Routing
Creating the physical connections between placed components following design rules.
